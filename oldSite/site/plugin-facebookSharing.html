Facebook Sharing plugin

This plugin allows your application to share content(links, photos, and videos) to Facebook within the App

Platforms:

only works with iOS 7+ and Android 5+

Gotchas:

you need Facebook App installed to share videos and photos, use canShow to check api

Functions:

facebookSharing.init()

-- initialize Facebook Sharing

facebookSharing.show(params, listener)

-- initialize Facebook Sharing

params(required)(table) {url=(string), photos=(array, array of pathForFile image files less than 12MB each), video=(file path, pathForFile video file less than 50MB), hashtag=(string, ex "#solar2d")}

(Note: for params you can have only have photos and video keys together and the url key should not use photos and video keys)

listener (function)

  • event.status
    • "did share"
    • "did cancel"
    • "error"
  • event.isError(boolean)
  • event.error(string)

facebookSharing.canShow(type)

-- returns boolean is can show

type (string) type can be "video", "photos", and "url"
