sceneKit.newParticleSystem()

Overview:

Creates a new blank particle system or one from a scene kit particle file

Syntax:

directory (string) (optional) path from root

fileName (string) (optional) name of particle file

Methods:

particle:setColor(red(number), green(number), blue(number), alpha(number) ) -- set color (0-1)

particle:setColorVariation(red(number), green(number), blue(number), alpha(number) ) -- set color variation(0-1)

particle:setImage(path)-- path(via system.pathToFile()) to image for particles

particle:addCollisonNode(node) add node whose geometry the system’s particles can collide with.

particle:removeCollisonNode(node) remove node whose geometry the system’s particles can collide with.

particle:remove()-- remove particles

Use these methods on nodes to add particles to any node type (newModel, newBox, newNode, etc)

node:addParticleSystem(particle), add particle obj(made via sceneKit.newParticleSystem())

node:removeParticleSystem(particle), remove particle obj(made via sceneKit.newParticleSystem())

node:removeAllParticleSystems(), remove all particle systems

Properties:

emissionDuration(number) -- The duration, in seconds, over which the system spawns new particles.

emissionDurationVariation(number) -- The range, in seconds, of randomized emission duration values.

idleDuration(number) -- The duration, in seconds, of periods when the system emits no particles.

idleDurationVariation(number) -- The range, in seconds, of randomized idle duration values.

loops(boolean) -- A Boolean value that determines whether the system repeats its emission and idle periods.

warmupDuration(number) -- The duration, in seconds, for which particles are spawned before the system is first rendered.

birthRate(number) -- The number of particles spawned during each emission period.

birthRateVariation(number) -- The range of randomized particle birth rate values.

emittingDirectionX(number) -- The possible initial directions for newly spawned particles, relative to the emitter shape x.

emittingDirectionY(number) -- The possible initial directions for newly spawned particles, relative to the emitter shape y.

emittingDirectionZ(number) -- The possible initial directions for newly spawned particles, relative to the emitter shape z.

spreadingAngle(number) -- The range, in degrees, of randomized initial particle directions.

particleAngle(number) -- The rotation angle, in degrees, of newly spawned particles.

particleAngleVariation(number) -- The range, in degrees of randomized initial particle angles.

particleVelocity(number) -- The initial speed, in units per second, for newly spawned particles.

particleVelocityVariation(number) -- The range, in units per second, of randomized initial particle speeds.

particleAngularVelocity(number) -- The initial spin rate, in degrees per second, of newly spawned particles.

particleAngularVelocityVariation(number) -- The range, in degrees per second, of randomized initial angular velocities for particles.

particleLifeSpan(number) -- The duration, in seconds, for which each particle is rendered before being removed from the scene.

particleLifeSpanVariation(number) -- The range, in seconds, of randomized particle life spans.

particleSize(number) -- The rendered size, in units of the scenes world coordinate space, of the particle image.

particleSizeVariation(number) -- The range of randomized particle sizes.

fresnelExponent(number) -- The reflectivity exponent SceneKit uses when rendering the particles image as a cube map.

stretchFactor(number) -- A multiplier for stretching particle images along their direction of motion.

imageSequenceRowCount(number) -- The number of rows for treating the particle image as a grid of animation frames.

imageSequenceColumnCount(number) -- The number of columns for treating the particle image as a grid of animation frames.

imageSequenceInitialFrame(number) -- The index of the first frame in a particle image animation.

imageSequenceInitialFrameVariation(number) -- The range of randomized initial frames for particle image animation.

imageSequenceFrameRate(number) -- The rate, in frames per second, of particle image animation.

imageSequenceFrameRateVariation(number) -- The range, in frames per second, of randomized frame rates for particle image animation.

imageSequenceAnimationMode(string)(default "repeat") -- modes "repeat"(The animation loops after displaying all of its images), "clamp"(The animation stops after displaying all of its images), and "autoReverse" (After the animation displays all of its images, it plays again in reverse order.)

affectedByGravity(boolean)(default false) -- A Boolean value that determines whether gravity, as defined by the scene’s physics simulation, affects the motion of particles.

particleDiesOnCollision(boolean)(default false) -- A Boolean value that determines whether particles are removed from the scene upon colliding with another object.

accelerationX(number) -- The constant acceleration vectodr, in units per second per second, applied to all particles in the system. affects x value

accelerationY(number) -- The constant acceleration vector, in units per second per second, applied to all particles in the system. affects y value

accelerationZ(number) -- The constant acceleration vector, in units per second per second, applied to all particles in the system. affects y value

dampingFactor(number) -- A factor that slows particles relative to their velocity.

particleMass(number) -- The mass, in kilograms, of each particle in the system.

particleMassVariation(number) -- The range, in kilograms, of randomized particle masses.

particleCharge(number) -- The electric charge, in coulombs, of each particle in the system.

particleChargeVariation(number) -- The range, in coulombs, of randomized particle charges.

particleBounce(number) -- The restitution coefficient of each particle in the system.

particleBounceVariation(number) -- The range of randomized restitution coefficients for particles.

particleFriction(number) -- The friction coefficient of each particle in the system.

particleFrictionVariation(number) -- The range of randomized friction coefficients for particles.

blendMode(string)(default "additive") -- values are "additive" (the source and destination colors are added together), "subtract"(The source color is subtracted from the destination color), "multiply"(The source color is multiplied by the destination color).

orientationMode(string)(default "billboardScreenAligned") -- values "billboardScreenAligned"(Each particles orientation is always fixed with respect to the point of view camera), "billboardViewAligned"(Each particle always faces the point of view camera (but may rotate about an axis parallel to the view direction)), "free"(Particle orientations are not restricted; they may rotate freely in all axes), and "billboardYAligned"(The y-axis direction of each particle is always fixed with respect to the point of view camera)

sortingMode(string)(default "none") -- values "none"(Particles are not sorted; they may be rendered in any order), "distance"(Particles farther from the point of view (as measured using distance from the camera in scene space) are rendered before closer particles), "oldestFirst"(Particles emitted earlier are rendered before particles emitted more recently), "youngestFirst"(Particles emitted more recently are rendered before particles emitted earlier), or "projectedDepth"(Particles farther from the point of view (as measured using projected depth) are rendered before closer particles.)

lightingEnabled(boolean) --A Boolean value that determines whether SceneKit applies lighting to particle images when rendering

blackPassEnabled(boolean) --A Boolean value that determines whether SceneKit renders particles in black before rendering the particle image
