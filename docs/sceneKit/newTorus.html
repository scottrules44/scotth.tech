sceneKit.newTorus()

Overview:

Creates a new sphere object that can be moved and adjusted

Syntax:

ringRadius (number) (required) The major radius of the torus, defining a circle in the x- and z-axis dimensions

pipeRadius (number) (required) The minor radius of the torus, defining the pipe that encircles the torus ring.

Methods:

inherents all other node methods

object:setColor(red(number), green(number), blue(number), alpha(number) ) -- set color (0-1)

object:setMaterial({image1(path), image2(path), image3(path), image4(path), ...}) -- image "paths" (see system.pathForFile) based on share more images will be need cube need 6 images.

object:setMaterialProperties(tableOfProperties)

Properties:

x(number) -- move object along x axis

y(number) -- move object along y axis

z(number) -- move object along z axis

xScale(number) -- how much the object should be scaled on the x axis (deafult 1)

yScale(number) -- how much the object should be scaled on the y axis (deafult 1)

zScale(number) -- how much the object should be scaled on the z axis (deafult 1)

rotationX(number) -- rotation around x axis in degrees

rotationY(number) -- rotation around y axis in degrees

rotationZ(number) -- rotation around z axis in degrees

castShadows(boolean) -- should object cast shadows

alpha(number) -- set transparency of object

ringRadius (number) The major radius of the torus, defining a circle in the x- and z-axis dimensions

pipeRadius (number) The minor radius of the torus, defining the pipe that encircles the torus ring.

ringSegmentCount (number) The number of subdivisions around the torus ring.

pipeSegmentCount (number) The number of subdivisions around the torus pipe.

nodeType(string) -- value "torus"
