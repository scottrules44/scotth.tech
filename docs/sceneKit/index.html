Scene Kit Plugin (Beta)

This plugin allows you make 3d games using apples scene kit library. Note this is a native 3d engine plugin not some 2d/2.5d plugin.

Pricing: I cannot prommise this will be free forever. I will probaly charge once this plugin is out of beta. This is a very big plugin which will require alot of maintenance and support.

Limitations:

Only works on iOS 8+ and tvOS 9+ (macOS 10.8+ hopefully coming soon, please request corona labs to link scene kit framework). Scene kit is write on top of Corona Open Gl. This mean what ever you display in Corona will be displayed behind all 3d objects.

Please Note: when node is said, I am referring to 3d scene kit nodes (newBox, newModel, newNode, etc).

How does the coordinate system work? Y+ = up , Y- = down, X+= right, X- = left, Z+ = closer to you, Z- equals far away. Please note 0,0 is the middle of the screen. Corona coords are different from scene kit coords

Functions:

  • sceneKit.init() --start up scene kit
  • sceneKit.deinit() --stop scene kit
  • sceneKit.autoLighting(boolean) --enable auto lighting(boolean true = on, false = off)
  • sceneKit.cameraControl(boolean) --enable camera control on objects(boolean true = on, false = off)
  • sceneKit.setFps() -- sets fps on scene kit
  • sceneKit.setBackgroundColor() --set color of scene
  • sceneKit.debugMode() --sets debug mode for scene kit to better see what is going on
  • sceneKit.cameraZ() -- sets cameras near and far depth limit
  • sceneKit.cameraFov() -- set cameras field of view, in degrees, on the vertical and horizontal axis
  • sceneKit.getCameraNode() --returns a node which can be positioned
  • sceneKit.newLight() --creates a new light source
  • sceneKit.newBox() --creates a new box/cubenode
  • sceneKit.newModel() --creates a new node from scene file
  • sceneKit.newFloor() --creates a new infinite plane node
  • sceneKit.newText() --creates a new 3d text node
  • sceneKit.newCone() --creates a new cone node
  • sceneKit.newCapsule() --creates a new capsule node
  • sceneKit.newCylinder() --creates a new cylinder node
  • sceneKit.newPyramid() --creates a new pyramid node
  • sceneKit.newSphere() --creates a new sphere node
  • sceneKit.newTorus() --creates a new torus node
  • sceneKit.newTube() --creates a new tube node
  • sceneKit.newPlane() --creates a new plane node
  • sceneKit.newNode() --creates a new blank node that can be used for things like particles or animations
  • sceneKit.newParticleSystem() --create a new particle system
  • sceneKit.addPhysics() --add physics to a node
  • sceneKit.setGravity() -- set gravity for physics
  • sceneKit.checkCollison() -- check for collison between two physics bodys
  • sceneKit.searchForNodeAtCoords() -- finds nodes at a corona point
  • sceneKit.moveSceneKitToFront() -- move scene kit to front
  • sceneKit.moveSceneKitToBack() -- move scene kit to back
