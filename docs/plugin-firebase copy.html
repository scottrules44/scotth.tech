Firebase helper

This plugin allows developers to easily handle the initialize firebase in one api call and call firebase methods from nodes. Note only the plugins listed here are supported

firebase.init()
-- initialize all supported firebase plugins included in build.settings

Nodes

note you must have the designated plugin in build.settings

firebase.storage
contains all functions of firebase storage (view docs)

firebase.analytics
contains all functions of firebase analytics (view docs)

firebase.config
contains all functions of firebase config (view docs)

firebase.database
contains all functions of firebase database (view docs)

firebase.messages
contains all functions of Corona Notfications v2 (view docs)

Build Settings:

You must include your google services plist and json in app root folder.
