Physics Body Editor Plugin

This plugins uses Physics Body Editor to edit shapes of objects in Corona Sdk.

Click here to learn how to install

Limitations:

Requires physics engine in Corona Sdk to use (to see those limitation click here). Also you have to install the Physics Body Editor.

Functions:

physicsBodyEditor.loadShape(name of object in json file, json file, object, [friction], [density], [bounce], [filter], [directory] )

name of object in json file

(required)(string) This is the name of the physics body you made in the Phyics body editor.

json file

(required)(string) This is what you named the file.

object

(required)(Display Object) the display object your want to give the physics shape to.

friction

(optional)(number) what you would like the friction to be on the physics body. If set to nil it will be .3

density

(optional)(number) what you would like the density to be on the physics body. If set to nil it will be 1

bounce

(optional)(number) what you would like the bounce to be on the physics body. If set to nil it will be .2

filter

(optional)(table) Filter values. To find out more check out Corona sdk's guide here

directory

(optional)(constant) the path of the json file, the default system.ResourceDirectory

Build.settings

settings =

{

	plugins =

	{

		["plugin.physicsBodyEditor"] =

		{

			publisherId="tech.scotth",

			marketplaceId = "insert marketplace account ID",

		},

	},

}
