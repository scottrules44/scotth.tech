Taptic Engine

This plugin uses the Taptic Engine this is will only work with iPhone 7 and 7 plus (and hopefully future devices)

Limitations:

Only works on iOS and "not" iOS simulator

Functions:

tapticEngine.vibrate(type)

type (optional) (string) heavy, light, and medium. deafult is medium

tapticEngine.set(type) --caches vibrate for reduced lag

type (optional) (string) heavy, light, and medium. deafult is medium

tapticEngine.destory() --removes vibrate cache

tapticEngine.activate() --vibrate cache

Build Settings:

settings =

{

	plugins =

	{

		["plugin.tapticEngine"] =

		{

			publisherId="tech.scotth",

			marketplaceId = "insert marketplace account ID",

		},

	},

}
