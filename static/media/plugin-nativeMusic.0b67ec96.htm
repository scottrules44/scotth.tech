Native Music Plugin

This plugin handle playback controls and create use native music apis for a Platfrom

Limitations:

Only works on iOS, tvOS, and MacOS

Functions:

nativeAudio.init(controlCallback)
--setups up control callbacks(remote events from control center and headphones)

^(Note the Native Audio Plugin handles all control events i.e when pause button is pressed from external source, the player will pause automatically )

controlCallback(function)(required) -- triggers event.status which can be "playPause"(triggers from play pause button i.e headphones), "pause", "play", "nextTrack"(tiggered from skip button), "previousTrack", and "songEnded"(triggers when song end naturally i.e not from hitting skip button )

nativeAudio.newTrack(fileParams)
--returns a track object
