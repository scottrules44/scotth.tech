Orientation Plugin

This plugin allows you to lock the orientation on your device

Platforms:

this plugin works on iOS and Android

Functions:

orientation.init()

orientation.lock(orientation)

-- orientation can be "all", "landscape", or "portrait"

Build Settings:

settings =

{

	plugins =

	{

		["plugin.orientation"] =

		{

			publisherId="tech.scotth",

			marketplaceId = "insert marketplace account ID",

		},

	},

}
