Passkit Plugin

This plugin allows you to add passes to the iOS wallet app (previously know as Passbook)

Platforms:

iOS 8+

Make Passes easy:

passkit

passslot

Functions:

passkit.add(filePath, listener)

file (path) path for pkpass file

listener (function) returns event.status = "showing" and "finished"
