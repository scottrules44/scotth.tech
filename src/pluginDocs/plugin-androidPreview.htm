Android Preview Plugin

This plugin allows you to preview files via the Android pop up.

Platforms:

this plugin only works on Android

Functions:

androidPreview.show(path)

path(string) path of file via system.pathForFile

Build Settings:

settings = 

{ 

	plugins = { 

		["plugin.androidPreview"] = 

		{ 

			publisherId="tech.scotth", 

			marketplaceId = "insert marketplace account ID", 

		}, 

	}, 

}
